# Contributing to Awesome AI Agents

Thank you for contributing to the Awesome AI Agents list!

## Adding an Agent

1. **Verify not listed** - Check README.md first
2. **Choose category** - Agents, Tools, Research, etc.
3. **Follow format**: `- [Agent Name](link) - Description`

## Categories

- Autonomous Agents
- Multi-Agent Systems
- Agent Frameworks
- Tools & Utilities
- Research Papers
- Tutorials & Guides

## Submission Guidelines

- Active, notable projects only
- Working URLs required
- Brief description (1-2 sentences max)
- No spam or self-promotion

## Pull Request Process

1. Fork the repo
2. Add to appropriate section in README.md
3. Submit PR with short description
