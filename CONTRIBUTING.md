# Contributing to Awesome AI Agents

Thanks for helping build the definitive list of AI agents!

## Quick Start

1. Fork this repository
2. Add your AI agent to the appropriate section
3. Open a Pull Request

## Adding a New AI Agent

### For Open Source Projects

Add to the [Open Source Projects](#open-source-projects) section:

```markdown
- [Agent Name](https://github.com/link) - One-line description
  - Stars: X · Forks: X · Contributors: X
  - Key feature 1
  - Key feature 2
  - Key feature 3
```

### For Closed-Source Products

Add to [Closed-source projects](#closed-source-projects-and-companies):

```markdown
| Name | Description | Category | Website |
|------|-------------|----------|---------|
| Product Name | One-line description | Category | [Link](url) |
```

## Quality Criteria

**Accepted:**
- ✅ Functional AI agents (not just concepts)
- ✅ Public demos or GitHub repos
- ✅ Clear documentation

**Not accepted:**
- ❌ Vaporware or announcement-only projects
- ❌ Dead projects (no activity in 12+ months)
- ❌ Generic AI wrappers without agent capabilities

## Categories

Open source agents should be organized by:
- **Frameworks** - Agent development frameworks
- **Coding agents** - Code generation/assistance
- **Research agents** - Information gathering
- **Automation agents** - Task automation
- **Multi-agent systems** - Agent orchestration

## Pull Request Checklist

- [ ] Agent added to correct section
- [ ] Description is clear and concise
- [ ] Links are working
- [ ] Following existing format/style
- [ ] Not a duplicate entry
