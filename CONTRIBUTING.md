# Contributing to Awesome AI Agents

Thank you for your interest in contributing! This document provides guidelines and instructions for contributing.

## How to Contribute

### Adding a New Agent

1. Fork this repository
2. Add your agent to the appropriate section in README.md
3. Follow the existing format:
   ``markdown
   - [Agent Name](link) - Brief description
   ``
4. Submit a pull request

### Guidelines

- **Quality over quantity**: Only add well-documented, actively maintained agents
- **Accurate descriptions**: Keep descriptions concise but informative
- **Proper categorization**: Place agents in the most relevant section
- **No duplicates**: Check if the agent is already listed

## Pull Request Process

1. Ensure your PR adds value to the collection
2. Follow the existing format and style
3. Wait for review and approval

## Code of Conduct

- Be respectful and constructive
- Focus on the content, not the person
- Help maintain a welcoming community

## Questions?

Open an issue if you have questions about contributing.
