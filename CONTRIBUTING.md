# Contributing to awesome-ai-agents

Thank you for your interest in contributing! This list curates resources, frameworks, and tools for building AI agents.

## How to Contribute

### Adding a New Entry

1. **Fork the repository** and clone it locally
2. **Add your entry** to the appropriate section in `README.md`
3. **Follow the format**: `- [Project Name](link) - Brief description`
4. **Verify relevance**: Only add AI agent-related resources
5. **Submit a Pull Request** with a clear description

### What to Include

- **Frameworks**: Agent frameworks, toolkits, and SDKs
- **Tools**: Agent development and deployment tools
- **Research**: Papers and research on AI agents
- **Applications**: Notable agent applications and demos
- **Learning**: Tutorials, courses, and educational content

### Guidelines

- **Quality over quantity**: Prefer well-maintained, popular projects
- **Clear descriptions**: Briefly explain what the resource offers
- **Alphabetical order**: Keep lists sorted within sections
- **Valid links**: Test all URLs before submitting
- **No duplicates**: Check that the resource isn't already listed

### Pull Request Checklist

- [ ] Entry follows the existing format
- [ ] Related to AI agents
- [ ] Project is actively maintained (if applicable)
- [ ] Added to the correct section
- [ ] No duplicate entries

## Code of Conduct

- Be respectful and inclusive
- Share knowledge constructively
- Welcome newcomers to the field

## Questions?

Open an issue for discussions or suggestions.
